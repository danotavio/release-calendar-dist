# Release Calendar — downloads

Aplicativo desktop para Windows 11 com o calendario de lancamentos de jogos (dados da IGDB).

**[Baixar a versao mais recente](../../releases/latest)** — `ReleaseCalendar-win-Setup.exe`.

O Windows pode exibir o aviso do SmartScreen (o instalador nao e assinado): clique em
"Mais informacoes" -> "Executar assim mesmo".

Este repositorio hospeda apenas os pacotes de instalacao e atualizacao. O codigo-fonte e privado.
