# Release Calendar 🎮

Aplicativo desktop para Windows 11 com um calendário mensal dos próximos lançamentos de videogames.
Dados da [IGDB](https://www.igdb.com/).

## ⬇️ Baixar e instalar

1. Baixe o **[ReleaseCalendar-win-Setup.exe](../../releases/latest)** (release mais recente).
2. Execute. O Windows pode mostrar **"O Windows protegeu o seu computador"** (SmartScreen) porque o
   instalador ainda não é assinado — clique em **Mais informações → Executar assim mesmo**.
3. Pronto. O app se instala e passa a se **atualizar sozinho** — você não precisa voltar aqui.

Prefere não instalar? Há também o **ReleaseCalendar-win-Portable.zip** (basta extrair e rodar o
`.exe`), mas a versão portátil não recebe atualizações automáticas.

## O que ele faz

- Calendário mensal navegável, com destaque de hoje e responsividade
- Detalhe do jogo: plataformas, notas da crítica e dos jogadores, franquia, jogos parecidos,
  screenshots, lojas e trailer embutido
- Favoritos, filtros por plataforma/gênero e notificações do Windows quando um favorito está perto de lançar
- Lembra posição, tamanho e o "sempre no topo" entre sessões

## Privacidade

Nenhuma conta, nenhum login. O app baixa os dados de lançamentos já prontos deste repositório (um
arquivo por mês, atualizado diariamente) — não fala com nenhuma API usando suas credenciais.

---

*Este repositório hospeda apenas os instaladores e os dados. O código-fonte é privado.*
